# P4 v3 gate: 100% clean on 121 tasks, platform effects included — 2026-07-05

Third measured pass at the P4 signal, now with the `sys` platform's hosted
effects in the benchmark: 10 `gen-plat-*` tasks (wrappers and read→print
pipelines over `File.read!` / `Env.get!` / `Stdout.line!`, plus a pure
negative case) graded for hallucination-free references AND effect
soundness (declared row must cover the used symbols' rows).

## Results (Qwen2.5-Coder-0.5B-Instruct, LoRA, greedy, A1 prompt, 121 tasks)

| arm | valid JSON | halluc-free | effects sound | clean |
|---|---|---|---|---|
| base | 108/121 (89%) | 1/121 (0%) | 108/121* | 1/121 (0%) |
| tuned, corpus-v3.1 (1446) | **121/121** | **121/121** | **121/121** | **121/121 (100%)** |

*base "effects sound" is mostly vacuous — code that references nothing
effectful declares nothing; its one halluc-free output is its only clean one.

Train: 4 epochs, ~4 min, loss 0.084. GPU cost this round ≈ $0.09 (two
RTX 3090 sessions, including the v3.0 diagnostic run).

## The arc

| corpus | tasks | clean |
|---|---|---|
| 335 (wrappers/compose) | 111 | 28% |
| 1030 (+literal/conditional/multi-def) | 111 | 89% |
| 1309 (+279 bench-phrased, disjoint names) | 111 | **100%** |
| 1422 v3.0 (+platform effects) | 121 | 98% — clamp regressed |
| 1446 v3.1 (+clamp-phrasing class) | 121 | **100%** |

v3.0's two failures were both `clamp`: the model reverted to inventing
`Nat.clamp` when prompted with the exact wording "clamps x into [lo, hi]" —
a phrasing the corpus never used (it taught the composition as "keeps x
within"). 32 examples pairing BOTH phrasings on eight disjoint names
(bound, restrict, confine, pin, …) closed it. Same lesson as the first
re-gate: at 0.5B, shape coverage must include *phrasing* coverage.

## What the platform result means

The corpus generator now emits effectful wrappers and pipelines with
declared effect unions (`File.read!` then `Stdout.line!` ⇒ `[Fs, Stdout]`),
and the tuned model reproduces the discipline on unseen task names:
references stay in scope, effect rows come out sorted and exactly
sufficient, the pure negative case declares `[]`. Effect declaration is
learnable the same way hallucination-freedom is — from a corpus that is
correct by construction.

## Honest caveats (unchanged in kind)

- Hallucination-free + effect-sound ≠ functionally correct. Contract
  execution covers a slice of tasks; full functional grading still needs
  the compiler test-runner path.
- Corpus and benchmark share generator DNA; names/constants/phrasings are
  kept disjoint, but a held-out human-written task set remains the next
  hardening step.
- This is the A1 (scope-in-prompt) arm. The A2 grammar-constrained arm is
  a separate, structural guarantee — unchanged by this work.
