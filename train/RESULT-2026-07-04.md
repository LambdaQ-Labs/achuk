# First bundled-model training run — 2026-07-04

The P4 cold-start pipeline, run end-to-end on real GPU.

## Setup
- **GPU:** RunPod RTX A5000 (24 GB), community cloud, $0.16/hr.
- **Base model:** Qwen2.5-Coder-0.5B-Instruct.
- **Corpus:** 335 synthetic examples from `claw corpus gen --stdlib` — every one references only real in-scope symbols, so the corpus cannot teach a hallucination.
- **Method:** LoRA SFT (r=16), 4 epochs, ~54s train time.
- **Total cost: $0.026.** (Balance $9.938 → $9.912.)

## Result
Training loss 0.90 → **0.27**. The fine-tuned 0.5B model emits **valid, in-scope Claw** it could not produce before:

```
Prompt: Define `sumTwo` : Nat, Nat -> Nat that adds its two arguments using in-scope `Nat.add`.
Tuned:  [{"name":"sumTwo","expr":{"Lam":{"params":["p0","p1"],
          "body":{"App":{"func":{"Var":"Nat.add"},"args":[{"Var":"p0"},{"Var":"p1"}]}}}},
          "ty":{"Fn":[[{"Named":"Nat"},{"Named":"Nat"}],{"Named":"Nat"}]}}]
```
Valid JSON, correct types, references `Nat.add` **by name** — no hallucinated symbols.

## What the run caught (and we fixed)
The *first* corpus referenced scope symbols by content hash (`{"Ref":"<hash>"}`). A model can reproduce a name but never guess a hash, so it hallucinated garbage hashes. Fix: the corpus now references symbols **by name** (`{"Var":"Nat.add"}`) — the same protocol the benchmark uses. Retrain after the fix drove loss to 0.27 and produced the clean output above.

## What this proves / doesn't
- **Proves:** the P4 machinery works — synthetic corpus → LoRA fine-tune → a model fluent in Claw's protocol, referencing only real symbols. The cold-start seed is real, and cheap ($0.03).
- **Doesn't yet prove:** the P4 *gate* (tuned-Claw ≥ stock-Python at repo-level). That needs a much larger corpus (more templates + real usage) and a full benchmark eval of the tuned model. This run establishes the pipeline; scaling the corpus + eval is the next step.

## Reproduce
```bash
# generate corpus, provision a GPU, then:
pip install -r requirements.txt trl==0.11.4 transformers==4.46.3 peft==0.13.2
python train.py --model Qwen/Qwen2.5-Coder-0.5B-Instruct --epochs 4
```
Adapter (~34 MB) is produced at `claw-lora/` (not committed; regenerate from corpus).
